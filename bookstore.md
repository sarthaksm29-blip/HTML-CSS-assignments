## üöÄ Features
* **Sticky Navigation**: The header remains fixed at the top, allowing users to navigate the site regardless of scroll depth.
* **Hover Effects**: Interactive book cards that lift upwards (`transform: translateY`) and change shadows when hovered.
* **Responsive Design**: Utilizes CSS **Flexbox** and `flex-wrap` to ensure the layout looks great on both desktops and tablets.
* **Modern UI Elements**: Custom-styled search bars, rounded buttons, and consistent padding/margins for a "breathable" design.
