# üì∏ Sarthakk Visuals Portfolio

A premium, dark-themed photography and fitness portfolio website. Designed with a minimalist aesthetic, this project focuses on high-impact visuals, smooth transitions, and a professional "luxury-brand" user experience.

---

## üåü Key Features

- **Cinematic Particle Engine:** A custom JavaScript-driven background featuring 40+ floating golden particles that create depth and motion in the header.
- **Interactive Lightbox Gallery:** Click on any photograph to view it in a distraction-free, full-screen modal.
- **Glassmorphic Navigation:** A sticky navigation bar with a real-time blur effect (`backdrop-filter`) for modern UI aesthetics.
- **Three Core Categories:**
    - **Gym:** High-intensity fitness and strength photography.
    - **Travel:** Scenic landscape shots and adventure visuals.
    - **Portraits:** High-contrast character and studio photography.
- **Responsive Layout:** Built with a flexible grid system that showcases images beautifully across different screen sizes.
